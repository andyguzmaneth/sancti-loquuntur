## Usage

1. Generate all images in Midjourney (or Nano Banana)
2. Remove backgrounds if needed (most should be clean cutout already)
3. Save to `images/` directory:
   - `images/augustine-hero.png` (hero, 3:4 ratio, ~1200x1600px)
   - `images/fresco-bg.jpg` (hero background, 16:9, ~2400x1350px)
   - `images/saints/augustine.png` (card, 1:1, ~600x600px)
   - `images/saints/aquinas.png`
   - `images/saints/bernard.png`
   - `images/saints/joan.png`
   - `images/saints/teresa.png`
   - `images/saints/john-cross.png`
   - `images/saints/catherine.png`
   - `images/saints/ignatius.png`
   - `images/saints/francis-sales.png`
   - `images/saints/therese.png`
4. Uncomment the `<img>` tags in `index.html` and remove placeholders
5. For the hero bg, uncomment the `background: url(...)` line in `.hero-bg`
